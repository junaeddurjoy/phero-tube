Discuss the scope of var, let, and const.
Students

Var: Var is varibale which is to define a variable type which can be an integer, string or any kind of variable.

Let: Let is to declare a variable which can be modified and reuse if needed. Also, let can be refered multiple and the main variable does not affect if the refered variable has been changed or modified.

Const: Const also uses to declare variable which also can be modified and reuse. But, unlike let it changes when a refered variable is changed.
